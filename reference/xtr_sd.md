# Standard Deviation

Standard Deviation

## Usage

``` r
xtr_sd(x, na_rm = FALSE)
```

## Arguments

- x:

  A numeric object of MCMC values.

- na_rm:

  A flag specifying whether to remove missing values.

## Value

A number.
