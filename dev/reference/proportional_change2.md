# Proportional Change2

Calculates the proportional change for a vector of two non-negative
numbers.

## Usage

``` r
proportional_change2(x)
```

## Arguments

- x:

  A numeric vector of length 2.

## Value

A number.

## See also

Other proportional fun2:
[`proportional_difference2()`](https://poissonconsulting.github.io/extras/dev/reference/proportional_difference2.md)

## Examples

``` r
proportional_change2(c(1, 2))
#> [1] 1
proportional_change2(c(2, 1))
#> [1] -0.5
```
