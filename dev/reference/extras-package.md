# extras: Helper Functions for Bayesian Analyses

Functions to 'numericise' 'R' objects (coerce to numeric objects),
summarise 'MCMC' (Monte Carlo Markov Chain) samples and calculate
deviance residuals as well as 'R' translations of some 'BUGS' (Bayesian
Using Gibbs Sampling), 'JAGS' (Just Another Gibbs Sampler), 'STAN' and
'TMB' (Template Model Builder) functions.
