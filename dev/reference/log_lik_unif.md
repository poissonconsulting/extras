# Uniform Log-Likelihood

Uniform Log-Likelihood

## Usage

``` r
log_lik_unif(x, min = 0, max = 1)
```

## Arguments

- x:

  A numeric vector of values.

- min:

  A numeric vector of the minimums.

- max:

  A numeric vector of the maximums.

## Value

An numeric vector of the corresponding log-likelihoods.
