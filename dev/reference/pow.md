# Power

R equivalent to the power function.

## Usage

``` r
pow(x, n)
```

## Arguments

- x:

  A numeric atomic object of the base.

- n:

  A numeric atomic object of the exponent.

## Value

A numeric atomic object of x raised to n.

## Details

Wrapper on `x^n`.
